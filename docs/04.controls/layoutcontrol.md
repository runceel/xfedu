# レイアウトコントロール

ここでは Xamarin.Forms でレイアウトに関係するコントロールについて紹介します。Xamarin.Forms では、以下のように様々なレイアウトコントロールが提供されています。

- StackLayout
- Grid
- AbsoluteLayout
- RelativeLayout
- ScrollView

## StackLayout
